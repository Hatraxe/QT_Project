Description détaillée

Gère les opérations de connexion et de déconnexion des utilisateurs.

Cette classe utilise UserStorage pour vérifier les informations d'identification des utilisateurs lors de la tentative de connexion et maintient une référence à l'utilisateur actuellement connecté.
