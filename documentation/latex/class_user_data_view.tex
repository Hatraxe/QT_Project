\doxysubsection{Description détaillée}
Classe représentant la vue des données de l\textquotesingle{}utilisateur. 

Fournit une interface graphique pour afficher les informations de l\textquotesingle{}utilisateur connecté et permettre certaines actions comme la déconnexion ou le changement de profil.
