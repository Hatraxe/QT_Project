\begin{DoxyReturn}{Renvoie}

Prend une liste d\textquotesingle{}objets \doxylink{class_user}{User}, les convertit en format JSON et les sauvegarde dans le fichier spécifié par filename. Si le fichier ne peut pas être ouvert ou si une erreur survient lors de l\textquotesingle{}écriture, la fonction retourne faux.


\begin{DoxyParams}{Paramètres}
{\em users} & La liste des utilisateurs à sérialiser. \\
\hline
{\em filename} & Le nom du fichier où sauvegarder le JSON. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Renvoie}
True si la sérialisation et la sauvegarde réussissent, False sinon.
\end{DoxyReturn}
Parcourt chaque utilisateur dans la liste fournie, extrait leurs informations et les convertit en format JSON. Ensuite, sauvegarde ce JSON dans le fichier spécifié par filename. Si le fichier ne peut être ouvert en écriture, retourne false.
