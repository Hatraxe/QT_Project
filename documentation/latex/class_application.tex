\doxysubsection{Description détaillée}
La classe \doxylink{class_application}{Application} représente la fenêtre principale de l\textquotesingle{}application. 

Cette classe gère la navigation entre les différentes vues de l\textquotesingle{}application et détient les instances principales de gestion des utilisateurs et de la connexion.
