 SQLite Database Viewer est une application développée en C++ avec le framework Qt. Elle permet de visualiser et d\textquotesingle{}interagir avec des bases de données SQLite. L\textquotesingle{}application offre une interface pour se connecter avec des profils d\textquotesingle{}utilisateurs différents, afficher et modifier les données des bases de données SQLite existantes.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Fonctionnalités}{Fonctionnalités}}\label{md__r_e_a_d_m_e_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Gestion des Utilisateurs}} \+: Création d\textquotesingle{}un nouvel utilisateur lors du premier lancement. Connexion/déconnexion avec mot de passe pour les utilisations suivantes.
\item {\bfseries{Profils Utilisateurs}} \+: Chaque utilisateur peut posséder plusieurs profils.
\item {\bfseries{Gestion des Droits}} \+: Les droits d\textquotesingle{}accès (visualisation, édition) sont spécifiques à chaque utilisateur.
\item {\bfseries{Super Utilisateur}} \+: Un super utilisateur par défaut ({\ttfamily email\+: superuser}, {\ttfamily mdp\+: password}) dispose de tous les droits, permettant une gestion avancée de l\textquotesingle{}application.
\item {\bfseries{Visualisation et Gestion des Bases de Données}} \+:
\begin{DoxyItemize}
\item Ajout et suppression de bases de données existantes via un {\ttfamily QFile\+Dialog}.
\item Sélection d\textquotesingle{}une base de données pour afficher ses tables et leurs contenus.
\item Exécution de requêtes SQL avec affichage des résultats.
\end{DoxyItemize}
\item {\bfseries{Sécurité et Gestion des Droits}} \+: Restriction des types de requêtes selon les droits de l\textquotesingle{}utilisateur (seul le super utilisateur peut exécuter toutes les requêtes).
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md__r_e_a_d_m_e_autotoc_md2}
